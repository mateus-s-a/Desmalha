Análise de Malhas

A Análise de Malhas é uma técnica usada para encontrar as correntes em um circuito planar. Baseia-se na Lei das Tensões de Kirchhoff (LTK), que afirma que a soma algébrica das tensões em qualquer laço fechado é igual a zero.

Passos:

  1. Identifique as malhas elementares.
  2. Atribua uma corrente de malha para cada malha (geralmente no sentido horário).
  3. Aplique a LTK para cada malha, expressando as tensões em função das correntes.
  4. Resolva o sistema de equações lineares resultante.

Análise Nodal

A Análise Nodal é usada para determinar a tensão em cada nó de um circuito em relação a um nó de referência (terra). Baseia-se na Lei das Correntes de Kirchhoff (LCK).

Passos:

  1. Identifique os nós principais do circuito.
  2. Escolha um nó de referência (Terra, V=0).
  3. Aplique a LCK para cada nó não-referência.
  4. Expresse as correntes em função das tensões nodais usando a Lei de Ohm.
  5. Resolva o sistema para encontrar as tensões.
